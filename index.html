About Pomodoro+

Pomodoro+ was created for a FreeCodeCamp exercise. While the exercise consisted of building a simple Pomodoro timer, I decided to add some features that I was interested in using. Specifically, I added the ability to track time and export timesheets. I also used the Notifications API in order to add desktop notifications for those times when sound-based notifications aren't appropriate.

You can see more of my work at htovey.com.
